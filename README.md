This project is a full-stack Productivity Tracker Chrome Extension designed to help users manage their time effectively while browsing the internet. The extension monitors the amount of time spent on different websites in real-time, allowing users to gain insights into their browsing habits and improve focus. It is built using a combination of modern web technologies including a Chrome Extension (frontend tracking layer), a Node.js and Express backend for handling data processing and storage, and MongoDB for persistent data management, making it a complete MERN stack application. The extension runs in the background and automatically tracks active tabs, calculates time spent on each domain, and stores this information locally using Chrome storage. Additionally, it includes a feature to block distracting websites such as social media platforms by redirecting them, helping users stay productive and avoid time-wasting activities. The popup interface provides a simple and user-friendly way to view tracked data, displaying a list of visited websites along with the time spent on each in seconds. For extended functionality, the backend server allows syncing of user data across devices, storing activity logs, and generating daily productivity reports. These reports can be accessed through a React-based dashboard that fetches data from the backend API and displays it in an organized format. The system is designed with scalability in mind, allowing future enhancements such as authentication, advanced analytics, productivity scoring, and visual charts using libraries like Chart.js. This project demonstrates practical implementation of browser APIs, asynchronous event handling, RESTful API development, and full-stack integration. It is ideal for students and developers looking to understand how Chrome extensions interact with web pages and how they can be combined with backend services for a richer experience. Overall, this productivity tracker provides a powerful yet simple solution for improving time management, increasing focus, and promoting better digital habits by giving users clear visibility into how they spend their time online.
